## 项目概述

本项目是一款基于 Python 和 OpenCV 的手势识别系统，包含了完整的源码、用户操作界面（UI）及使用教程。

## 技术栈

- **后端：**
  - Python 3.x
  - OpenCV 库
- **前端：**
  - Tkinter（或其他图形界面库）

## 功能模块

- **手势识别核心算法**
- **用户界面：**
  - 实时视频流显示
  - 手势识别结果展示
- **教程与文档：**
  - 使用指南
  - 开发文档

## 系统角色

- 用户：通过 UI 进行手势识别操作

## 运行环境

- 操作系统：兼容 Windows、Linux、MacOS
- Python 环境配置：
  - Python 3.x
  - 安装有 OpenCV 库

## 部署步骤

1. 安装 Python 3.x
2. 安装 OpenCV 库
3. 下载源码压缩包
4. 解压并进入项目目录
5. 执行 `python main.py` 启动程序
